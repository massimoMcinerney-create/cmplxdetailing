# CMPLX Detailing — Cinematic Web Experience

A high-end interactive website concept for a luxury automotive detailing brand built using modern web animation tools.

This project focuses on creating a **cinematic, scroll-driven 3D experience** where a cherry-red Corvette C6 is displayed in a floating studio environment and transforms through a full detailing process as the user scrolls.

---

## 🚗 Concept

Instead of a traditional website, this is a **scroll-controlled CGI-style experience** inspired by:

- Apple product pages
- Tesla vehicle showcases
- BMW configurator visuals
- Luxury automotive commercials

The car is presented as a floating centerpiece in a dark cinematic studio environment.

As the user scrolls, the car transitions through:

1. Pre-wash (dirt removal & water simulation)
2. Foam bath (full coverage cinematic foam layer)
3. Contact wash (hand wash illusion with reflections restoring)
4. Dry process (water bead removal + final polish)
5. Final reveal (fully detailed showroom finish)

---

## 🧠 Tech Stack

- Three.js (3D rendering)
- GSAP + ScrollTrigger (scroll-based animation timeline)
- JavaScript (core logic)
- HTML/CSS (layout + UI layer)

---

## 🎯 Goal

To simulate a premium, cinematic automotive experience inside the browser using:
- Scroll-driven animation systems
- Camera movement + lighting changes
- Shader-based visual illusions
- Real-time 3D scene manipulation

The focus is not on complex simulation accuracy, but on **visual realism and immersive feel**.

---

## ⚙️ Features

- Floating 3D Corvette scene
- Scroll-controlled camera movement
- Dynamic lighting transitions per stage
- Cinematic atmosphere (fog, reflections, shadows)
- Detailing progression system (dirty → showroom clean)
- Minimal, luxury UI overlay
